## 🔒 Summary of What Miko Collects

| Data | Purpose | Stored? |
|---|---|---|
| User IDs | Command targeting, warns, AFK | ✅ Yes |
| Server IDs | Per-server config, logs, prefix | ✅ Yes |
| Channel/Role IDs | Autorole, log channels | ✅ Yes |
| Message Content | Snipe & AFK detection only | ❌ Temporary |
| IGN / Tier Data | Ranking features via Firebase | ✅ Yes |
| Warn Records | Moderation history | ✅ Yes |

**We do NOT sell or share your data. Ever.**

## 🌐 GitHub Pages Setup

This policy is hosted via **GitHub Pages** from the `main` branch root.

To enable after pushing:
1. Go to repo **Settings → Pages**
2. Set source to **Deploy from a branch → main → / (root)**
3. Save — your site will be live at `https://MikoDev-Sd.github.io/miko-privacy/`
